# 구글 태그 메니저 (GTAG)
- 2012년 가을에 출시한 서비스
- 인터넷상의 인터페이스로부터 웹분석, 광고성과 측정, 제휴 마케팅 추적등 다양한 태그를 관리 할 수 있도록 하는 솔루션
- 태그를 관리한다.
