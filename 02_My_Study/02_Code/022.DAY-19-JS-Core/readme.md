#hasOwnProperty
모든 객체는 hasOwnProperty 를 상속하는 Object의 자식이다. 이 메소드는 객체가 특정 프로퍼티를 자기만의 직접적인 프로퍼티로서 소유하고 있는지를 판단하는데 사용된다.
**in 연산과는 다르게, 이 메소드는 객체의 프로토타입 체인을 확인하지는 않는다.**
